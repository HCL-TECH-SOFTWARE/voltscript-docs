      • FindObjectByPath

        Function FindObjectByPath(Labels) As XMLObject
        Finds an XMLObject within the current XMLObject using an array of labels to navigate down the hierarchy.
        idx Argument Data Type Attributes Comments
        1LabelsArrayArray of labels to use to find the XMLObject 
         
        Code Sample
        %REM
            The following example loads a string of XML and finds the element <John>
        %END REM
        Dim parser as New XMLParser
        Dim object as XMLObject
        Dim childObj as XMLObject
        Dim path(2) as String
        Call parser.loadFromXML("<Profile version='1' publish='2024'><Employee><John>Doe</John></Employee></Profile>")
        Set object = parser.getRootXML()
        path(0) = "Profile"
        path(1) = "Employee"
        path(2) = "John"
        Set childObj = object.findObjectByPath(path)
        Print childObj.toString(True)

      • ReplaceChildObject

        Function ReplaceChildObject(FirstObject As XMLObject, SecondObject As XMLObject) As XMLObject
        Replaces the first Child Object argument from the current XMLObject using the second Child Object argument.
        idx Argument Data Type Attributes Comments
        1FirstObjectXMLObject 
        2SecondObjectXMLObject 
         
        Code Sample
        %REM
            The following example loads a string of XML and replaces the element <Employee> to <Foo>
        %END REM
        DIm object as New XMLObject
        Dim parser as New XMLParser
        Call parser.loadFromXML("<Profile version='1' publish='2024'><Employee>John Doe</Employee></Profile>")
        Set object = parser.getRootXML()
        DIm OldChild as New XMLObject, NewChild as New XMLObject
        Set OldChild = object.getChild("Employee")
        Call NewChild.addRootTag("Foo","Bar")
        Call object.ReplaceChildObject(OldChild,NewChild)
        Print object.toString(True)
