      • GetRowArray

        Function GetRowArray(Skip As Integer, Limit As Integer) As String Array
        Allows iteration through a CouchView, returning a selected set of rows of information.

        Returns String Array, with each element contining the JSON for the specific row it represents.
        idx Argument Data Type Attributes Comments
        1SkipInteger-UnsignedNumber of rows to SKIP prior to retrieving requested rows.  
        2LimitInteger-UnsignedBY_VALMaximum number of rows to return.  
         
        Code Sample
        ' Sample code assumes previously instantiated view as CouchView object 
         
        Dim rows As Variant 
         
        rows = view.getRowArray(0, 10)
        Print |rows: | & join(rows, |, |) 
         
        %REM sample view contains 3 documents, Print statement returns: 
        	rows: {"id":"Doc 3","key":"boiled water","value":"Bob.3"}, {"id":"Doc 2","key":"nutella/jelly","value":"Bob.2"}, {"id":"Doc 1","key":"pasta","value":"Bob.1"}
        %END REM

      • GetRowsByKey

        Function GetRowsByKey(StartKey As String, EndKey As String, Limit As Integer) As String
        Retrieves a specific subset of rows from the CouchView.

        Returns a JSON String containing the content of all rows in the result set.
        idx Argument Data Type Attributes Comments
        1StartKeyStringKey specifying the FIRST row to retrieve 
        2EndKeyStringKey specifying the LAST row to retrieve 
        3LimitInteger-UnsignedOPTIONAL,
        BY_VAL,
        OptionalValue: 25        		Maximum number of rows to retrieve 
         
        Code Sample
        ' Sample code assumes previously instantiated view as CouchView object 
         
        Dim rows As String 
         
        rows = view.getRowsByKey("boiled water", "nutella/jelly", 10)
        Print |rows: | & rows 
         
        %REM sample view contains 3 documents, but only 2 match the specified keys, Print statement returns: 
        	rows: {"total_rows":3,"offset":0,"rows":[{"id":"Doc 3","key":"boiled water","value":"Bob.3"},{"id":"Doc 2","key":"nutella/jelly","value":"Bob.2"}]}
        %END REM
