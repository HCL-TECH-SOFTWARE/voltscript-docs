      • IsModified (Get)

        Property Get IsModified As Boolean
        Flag Indicating whether or not the in-memory CouchDocument has been modified since last saved.

        Behavior Notes:
        1) This will always return False for Newly created and not-yet-saved CouchDocument objects.
        2) If a CouchDocument has been saved (or newly retrieved from disk) and a CouchAttachment is added, this will return False.
        3) If a CouchAttachment has been uploaded to a CouchDocument , this will return False.
        4) If a new item is added to a CouchDocument, this will return True.
         
        Code Sample
        ' Sample code assumes previously created document As CouchDocument object 
         
        Print |document.isModified: | & document.isModified

      • IsSaved (Get)

        Property Get IsSaved As Boolean
        Flag indicating if an in-memory CouchDocument has been saved since last modified.

        Behavior Notes:
        1) This will always return False for Newly created and not-yet-saved CouchDocument objects.
        2) If a CouchDocument has been saved (or newly retrieved from disk) and a CouchAttachment is added, this will return True.
        3) If a CouchAttachment has been uploaded to a CouchDocument , this will return True.
        4) If a new item is added to a CouchDocument, this will return False.
         
        Code Sample
        ' Sample code assumes previously created document As CouchDocument object

      • Revision (Get)

        Property Get Revision As String
        Unique code representing the revision value of the on-disk CouchDocument . This value is composed of 2 parts delimited by a hyphen. This value is generated by Apache CouchDB itself, and if officially "opaque" - other than it is unique.

        Behavior Notes:
        1) This is not set until the CouchDocument has been saved.
        2) If a CouchDocument has been saved, and a CouchAttachment is added to it, this value does not change until the CouchAttachment has been Uploaded.
        3) This value will change whenever a CouchAttachment is Uploaded
         
        Code Sample
        %REM 
          Sample code assumes previously created document As CouchDocument object, with the JSON Value of: 
        {"$$Meta$$":{"$LastModified":"2023-09-23T13:05:26Z"},"_id":"76796ba0df26f8c2e539cf39d300a38a","_rev":"1-4134b4b21a14e6718782af9466ad32c3","recipe":{"CookTime":5,"Difficulty":"","Directions":["Cube 6 cups of watermelon","Stir in olive oil and lime juice","Chop, and stir in feta cheese","Garnish with fresh basil"],"Ingredients":["6 cups cubed watermelon","2 tbsp. olive oil","1 tbsp. lime juice","1/3 cup chopped feta cheese","1/3 cup basil leaves"],"PrepTime":15,"Servings":4,"name":"Watermelon Salad"}}
        %END REM 
        Print |document.Revision: | & document.Revison 
        ' Prints "document.Revision: 1-4134b4b21a14e6718782af9466ad32c3"

        Function CreateAttachment(Name As String, DiskFile As String, ContentType As String) As CouchAttachment
        Creates a CouchAttachment object.
        IMPORTANT NOTE: The newly created CouchAttachment object exists only in memory. It must be uploaded using CouchAttachment.Upload() method in order to be saved to the Couch Database.
        idx Argument Data Type Attributes Comments
        1NameString 
        2DiskFileString 
        3ContentTypeString 
         
        Code Sample
        ' Sample code assumes previously created document As CouchDocument object 
         
        Dim attachment As CouchAttachment 
        Dim uploadedbytes As Long
         
        Set attachment = document.createAttachment("watermelonsalad.png", CurDir() & "/resources/watermelonsalad.png", "image/png")
        Print |Created Attachment|
        Print |attachment.Name: | & attachment.Name 
        Print |attachment.ContentLength: | & attachment.ContentLength 
        Print |attachment.ContentType: | & attachment.ContentType 
        Print |attachment.DiskFile: | & attachment.DiskFile 
        Print ||        
        Print |Attachment Added but Not Yet Uploaded|
        Print |document.JSONValue: | & document.JSONValue        
        Print || 
         
        ' !! Document must be saved to disk before uploading attachments !! 
        If Not document.isSaved Then Call document.Save() 
         
        uploadedbytes = attachment.Upload()
        Print |attachment.Upload() returned | & uploadedbytes & | bytes.|
