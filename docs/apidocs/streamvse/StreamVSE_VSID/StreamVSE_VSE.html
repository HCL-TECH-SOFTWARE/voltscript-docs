VSE StreamVSE

      • StreamVSE.StreamVSE

  • Represents a file as a stream of binary or character data.
    Inspired by the exisiting NotesStream class from LotusScript. Provides an interface to a file, allowing the user to read from a file or write new data to the end. Supports both raw byte operations as well as common text encodings, including UTF-8 and UTF-16. Text can be read line by line or all at once. Automatically closes the stream when objects of this class are destroyed.
