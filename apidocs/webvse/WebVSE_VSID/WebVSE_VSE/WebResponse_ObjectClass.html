      • AllCookies (Get)

        Property Get AllCookies As String Array
        Retrieves all cookies as an array of strings, where the value is a tab-separated cookie containing seven fields:
        - Domain name
        - Include subdomains
        - Path
        - Set over a secure transport
        - Expires at, either seconds since Jan 1st 1970 or 0
        - Name of the cookie
        - Value of the cookie
         
        Code Sample
        %REM
            The following code creates a WebServer and WebRequest, sends the request and prints the cookies in the WebResponse.
        %END REM
        Dim server as New WebServer()
        Dim request as WebRequest
        Dim response as WebResponse
        server.protocol = "HTTPS"
        server.HostURL = "httpstat.us"
        Set request = server.createRequest()
        request.target = "200"
        request.timeoutSeconds = 5
        Set response = request.send()
        ForAll cookie in response.AllCookies
            Print "Cookie is " & cookie
        End ForAll
