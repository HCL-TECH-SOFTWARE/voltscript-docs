        languageErrors = False

            if (error.code < 1000) Then languageErrors = True
        End ForAll

      • createCustomErrorEntry

        Public Function createCustomErrorEntry(message As String, code As Integer, lineNum As Integer, levelNum As Integer) As ErrorEntry
        Spawns a new Custom ErrorEntry object and adds it to the ErrorSession. This can be used instead of a Try/Catch. Conditionally spawns a LogEntry in the Global LogSession instance.
        idx Argument Data Type Attributes Comments
        1messageStringError message describing the error. If code is less than 1 the value of Error$() will be used. 
        2codeIntegerNumeric code of the error. If less than 1 the value of Err() will be used.  
        3lineNumIntegerLine number in the source code where the error occurred. If code is less than 1 the value of Erl() will be used.  
        4levelNumIntegerLogging Level (NO_LOGGING, LOG_TRACE, LOG_DEBUG, etc). Used for conditionally creating a LogEntry  
         
        Code Sample
        If response.ResponseCode != 200 Then
            Call getErrorSession().createCustomErrorEntity("Bad response: " & response.ResponseCode, 1400, 275, LOG_WARN)
            Return False
        End If
