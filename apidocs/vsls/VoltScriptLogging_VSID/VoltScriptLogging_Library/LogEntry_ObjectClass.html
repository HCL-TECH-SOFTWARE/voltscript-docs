      • extInfo (Get)

        Public Property Get extInfo As String
        Extended info that may be included with the message, such as debug info, stack trace, etc. Set in the object's constructor.

      • level (Get)

        Public Property Get level As Integer
        The logging level assigned to the object. Specified as an argument in the object's constructor.

      • levelName (Get)

        Public Property Get levelName As String
        Name of the logging level for the object.

      • libraryName (Get)

        Public Property Get libraryName As String
        Name of the VSS library where the object creation was triggered.

      • lineNum (Get)

        Public Property Get lineNum As Integer
        Line number of the running code where VoltScriptLogging was called.

      • message (Get)

        Public Property Get message As String
        The text message of the object (passed as an argument to the object's constructor).

      • methodName (Get)

        Public Property Get methodName As String
        Name of the method (Property, Event, Sub, or Function) where the object creation was triggered.

      • stackTrace (Get)

        Public Property Get stackTrace As String
        Stack Trace of the object creation, delimited by LF, in the format of "libraryName, methodName, linenum" for each line in the stack trace.

      • timeStamp (Get)

        Public Property Get timeStamp As String
        Timestamp of when the object instance was created. This is date/time in yyyy-MM-dd hh:mm:ss format in the current platform's timezone.
