      • suppressErrors (Get)

        Public Property Get suppressErrors As Boolean
        Suppress errors on adding elements. Errors wll still be thrown if the Collection is locked
        or attempting to insert an entry into a sorted Collection.
        Instead of throwing an error, e.g. for wrong content type, the method will just return false
         
        Code Sample
        %REM
        The following code will print "Error adding" for the first Try/Catch. But for the second, the error will be suppressed and the call to .add() will return False
        %END REM
        Dim coll as New Collection("STRING", Nothing, False, False)
        Dim success as Boolean
        Try
            success = coll.add(2)
            Print success    ' Code will not get to this line
        Catch
            Print "Error adding"   ' This will be printed
        End Try
        coll.suppressErrors = True
        Try
            success = coll.add(2)
            Print success    ' Code will print False
        Catch
            Print "Error adding"   ' This will NOT be printed
        End Try

        Code Sample
        %REM
        The following code shows how to create a transformer to square values. The code is very basic with no error handling.
