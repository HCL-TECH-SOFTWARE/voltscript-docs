      • Revision (Get)

        Property Get Revision As String
        Unique code representing the revision value of the on-disk CouchAgent . This value is composed of 2 parts delimited by a hyphen. This value is generated by Apache CouchDB itself, and if officially "opaque" - other than it is unique.

        Behavior Notes:
        1) This is not set until the CouchAgent has been saved.

        Function ImportAgentScript(FileName As String, IsURL As Boolean) As Boolean
        Imports source code into the CouchAgent
        idx Argument Data Type Attributes Comments
        1FileNameString 
        2IsURLBooleanBY_VAL 
         
        Code Sample
        ' Sample code assumes previously instantiated database as CouchDatabase object and a valid value for viewname 
         
        Dim rows As String 
        Dim agent As CouchAgent 
         
        Set agent  = database.createAgent(agentname)
        Print |Spawned CouchAgent|
         
        agent.AgentLanguage = |VoltScript|
        agent.AgentMetadata = |{"Author":"James Bond","Environment":"Casino Royale"}|
         
        %REM Sample Agent Agent007.vss source code:
        	Option Public
        	Option Declare 
         	  
        	Sub Initialize
        		MessageBox |Always the gentleman, the first drink I ever ordered was an Americano Cocktail, which constists of 1 measure Campari, 1 measure sweet vermouth, Soda water. (Casino Royale)|
        	End Sub     
        %END REM 
         
        Print |Importing Agent Code|       
        Call agent.ImportAgentScript(CurDir() & |/src/Agent007.vss|, False) 
        Call agent.Save() 
         
        Print |Executing Agent Code|
        Execute agent.AgentCode 
        Print || 
         
         
        %REM Prints
        	Spawned CouchAgent
        	Importing Agent Code
        	Executing Agent Code

        %END REM
