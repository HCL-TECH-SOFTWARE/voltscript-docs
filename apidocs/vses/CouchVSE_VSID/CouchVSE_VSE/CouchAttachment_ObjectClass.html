Public Class CouchAttachment

      • CouchVSE.CouchAttachment

  • Carrier for Meta information about an attached file for a CouchDocument 
     
    Creation - Use the CouchDocument.CreateAttachment() method 
    Retrieval - Use the CouchDocument.GetAttachmentInfo() method.

        Downloads the file for the CouchAttachment object to the LOCAL filesystem.
        idx Argument Data Type Attributes Comments
        1DiskFileString 
        2OverwriteBooleanBY_VAL 
         
        Code Sample
        ' Sample code assumes previously instantiated document as CouchDocument object 
           
        Dim attachment as CouchAttachment 
        Dim bytes As Long 
        Dim targetfilepath As String 
         
        Set attachment = document.getAttachmentInfo("watermelonsalad.png")
        Print |acquired attachment for "watermelonsalad.png"|
         
        targetfilepath = CurDir() & "/resources/watermelonsalad_copy.png"
         
        Print |downloading to | & targetfilepath 
        bytes = attachment.Download(targetfilepath, True)
         
        Print |Attachment.Download() returned | cstr(bytes)

      • Upload

        Function Upload() As Long
        Uploads the LOCAL filesystem file to the ParentDatabase of the CouchAttachment.
         
        Code Sample
        ' Sample code assumes previously instantiated document as CouchDocument object 
           
        Dim attachment As CouchAttachment 
        Dim uploadedbytes As Long
         
        Set attachment = document.createAttachment(|watermelonsalad.png|, CurDir() & |/resources/watermelonsalad.png|, "image/png")
        Print |Created Attachment|
        Print |attachment.Name: | & attachment.Name 
        Print |attachment.ContentLength: | & attachment.ContentLength 
        Print |attachment.ContentType: | & attachment.ContentType 
        Print |attachment.DiskFile: | & attachment.DiskFile 
        Print ||        
        Print |Attachment Added but Not Yet Uploaded|
        Print |document.JSONValue: | & document.JSONValue        
        Print || 
         
        ' !! Document must be saved to disk before uploading attachments !! 
        If Not document.isSaved Then Call document.Save() 
         
        uploadedbytes = attachment.Upload()
        Print |attachment.Upload() returned | & uploadedbytes & | bytes.|
        Print ||
