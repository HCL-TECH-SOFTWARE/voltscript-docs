      • ExtractFile

        Function ExtractFile(DestinationDirectory As String, CreateDirectory As Boolean, OverwriteFile As Boolean) As Boolean
        Extracts the file to a directory, with options to create the directory and overwrite the file.
        idx Argument Data Type Attributes Comments
        1DestinationDirectoryStringDirectory into which to extract the file 
        2CreateDirectoryBooleanBY_VALWhether to create the destination directory 
        3OverwriteFileBooleanBY_VALWhether to overwrite if the destination includes a file with the same name. If false, the file will not be extracted. 
         
        Code Sample
        %REM
            The following code Extracts top-level files only, putting them in the current directory
        %END REM
        Dim zip as New ZipArchive()
        Call zip.openArchive(CurDir & "/zip1.zip")
        ForAll zipFile in zip
           If Not zipFile.isDirectory Then zipFile.extractFile(CurDir, False, True)
        End ForAll
