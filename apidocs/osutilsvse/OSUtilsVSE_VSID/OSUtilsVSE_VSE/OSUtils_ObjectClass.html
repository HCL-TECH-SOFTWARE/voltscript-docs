        Function GetEnvironment(EnvVar As String) As String
        Gets an environment variable. OSUtils uses a snapshot of environment variables that is separate to the snapshot used by the core runtime.
        So if you set an environment variable using OSUtils, you must use OSUtils to retrieve the updated value.
        idx Argument Data Type Attributes Comments
        1EnvVarStringEnvironment variable 
         
        Code Sample
        %REM
            Getting the home directory the "harder" way. Just use OSutils.homeDir
        %END REM
        Dim OSUtils as New OSUtils()
        Dim platform as String
        Dim envVar as String
        Dim homeDir as String
         
        platform = OSUtils.platform
        Select Case platform
        Case "Windows64":
            envVar = "USERPROFILE"
        Case "LINUX":
            envVar = "HOME"
        Case Else
            Error 1001, "Unexpected platform - " & platform
        End Select
         
        Print OSUtils.getEnvironment(envVar)

      • MakeDirectories

        Function MakeDirectories(Path As String) As Boolean
        Conditionally attempts to create a directory in the filesystem.

        Checks for the existence of a directory, and if it does not exist attempts to create it. Multiple level directories are supported.

        @return Flag indicating success / failure. True indicates either the directory already exists or was successfully created.
        idx Argument Data Type Attributes Comments
        1PathStringPath to create directory, e.g. /temp/foo/bar.  
         
        Code Sample
        %REM
            The following cde creates directories
        %END REM 
          Dim myOS as New OSUtils()
          Dim myresult as Boolean
        ' assuming directory /temp exists and contains a single subfolder "test1" 
          myresult = myOS.makeDirectories("/temp/test1")  
          ' myresult is true because  "test1" already exists within "temp"
          myresult = myOS.makeDirectories("/temp/foo/bar") 
          ' myresult is true if folder "foo" was successfully created within "temp", AND folder "bar" was successfully created witihin "foo".  
          myresult = myOS.makeDirectories("/temp/jabberwocky") 
          ' if a failure occurred (such as the user not having the right to create filesystem directories) then myresult would be false.
