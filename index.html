    Welcome to VoltScript documentation

    VoltScript is a BASIC scripting language evolved from LotusScript®, which was developed for the Lotus Software family of products. The language has been extended for use with HCL Volt MX Go, as a server-side scripting language running within the Volt Foundry middleware layer.
